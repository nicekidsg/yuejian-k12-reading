# 阅见英语

面向中国 K12 儿童的英语原著推荐、词汇学习、听书与阅读成长平台。

## 在线体验

[https://yuejian-k12-reading.leyangsh.chatgpt.site](https://yuejian-k12-reading.leyangsh.chatgpt.site)

## 主要功能

- 280 本免费英文儿童读物：完整纳入 200 本精选书单，并保留原有优质书目
- 80+ 本适合 4–7 岁亲子共读或启蒙阅读的公版绘本、童谣与短篇
- 逐页阅读、页码与适龄提示；优先穿插原版公版配图，对缺图的低龄读物补充明确标注的 AI 儿童插图
- 当页重点词汇总数、释义、音标和发音；点击词卡可高亮正文中的对应单词
- 公开有声书优先，缺失时使用浏览器文字转语音朗读
- 收藏、阅读进度、连续打卡、星图成长记录和好友进度

## 本地开发

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

## 数据与版权说明

书目、正文、原版配图和音频以公共领域或公开可用来源为主，来源信息随书展示。AI 生成插图在阅读器内单独标注，不与原版图片混淆。正式商业化前，请再次核对各资源在目标地区和具体使用场景下的授权条款。
