# 阅见英语

面向中国 K12 儿童的英语原著推荐、词汇学习、听书与阅读成长平台。

## 在线体验

[https://yuejian-k12-reading.leyangsh.chatgpt.site](https://yuejian-k12-reading.leyangsh.chatgpt.site)

## 主要功能

- 100 本免费英文儿童读物，支持按年龄、难度和主题探索
- 逐页阅读、页码与适龄提示，并穿插公开授权或公共领域配图
- 当页重点词汇释义、音标和发音；点击词卡可高亮正文中的对应单词
- 公开有声书优先，缺失时使用浏览器文字转语音朗读
- 收藏、阅读进度、连续打卡与星图成长记录
- 好友系统，可查看好友公开的打卡和阅读进度

## 本地开发

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```
